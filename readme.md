[![Build Status](https://travis-ci.org/laravel/framework.svg)](https://travis-ci.org/laravel/framework)


## Installation:
1. composer update
2. php artisan key:generate
3. touch database/database.sqlite
4. [sudo] npm install
5. npm install laravel-elixir
6. php artisan migrate
7. gulp
8. php artisan serve
